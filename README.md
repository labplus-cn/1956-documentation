## 编译

本地编译方法:

    pip3 install -r requirements.txt

    git clone https://github.com/labplus-cn/1956-documentation.git

    cd documentation

    make html

如果成功，构建的HTML文件将位于其中documentation/_build 文件夹，
HTML 页面保存在 _build/html 目录。
